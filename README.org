* Usage

* Install

Efetch use makefile for installation.

#+BEGIN_EXAMPLE
make install
#+END_EXAMPLE

** Upgrade

#+BEGIN_EXAMPLE
make upgrade
#+END_EXAMPLE

** Uninstall

#+BEGIN_EXAMPLE
make uninstall
#+END_EXAMPLE
